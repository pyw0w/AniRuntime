# AniRuntime

Публичный запускатель для платформы AniCore.

## Установка

```bash
go get github.com/pyw0w/AniRuntime@latest
```

## Использование

```bash
go run github.com/pyw0w/AniRuntime/cmd/runtime -config config.json
```

## Конфигурация

Скопируйте `config.example.json` в `config.json` и настройте под свои нужды.

## Зависимости

- `github.com/pyw0w/AniApi` - публичный API
- `github.com/pyw0w/AniCore` - приватное ядро (требует доступа)
